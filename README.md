# Animation in Virtual Environments Projects
### Interação em Ambientes Virtuais – ISEL

This repository contains the practical assignments and final project developed for the course **Animation in Virtual Environments**.

Throughout this course, I worked with:

- Unity
- C#
- Procedural World Generation
- Perlin Noise & Fractal Brownian Motion (fBM)
- Mesh Generation
- Unity ML-Agents
- Reinforcement Learning (PPO)
- Python
- MediaPipe
- Computer Vision
- UDP Networking
- Gesture Recognition
- Human-Computer Interaction (HCI)

# Practical Assignments

---

## Practical Work 1 – Procedural Minecraft World

Development of a procedurally generated voxel world inspired by Minecraft using Unity.

### Main Features

- Procedural terrain generation
- Perlin Noise terrain shaping
- Chunk-based world generation
- Automatic tree generation
- Procedural lakes
- Randomized ore distribution
- fBM-based mineral placement
- Modular world generation system

### Focus

Procedural generation, terrain algorithms, Unity world creation and optimization.

---

## Practical Work 2 – Reinforcement Learning with ML-Agents

Development of an intelligent agent capable of learning how to match RGB colors through reinforcement learning.

### Technologies

- Unity ML-Agents
- PPO (Proximal Policy Optimization)
- Reinforcement Learning
- Python training pipeline

### Main Features

- Custom training environment
- RGB color matching task
- Reward and penalty system
- Observation space design
- PPO hyperparameter tuning
- Training statistics analysis

### Focus

Artificial Intelligence, reinforcement learning, agent training and environment design.

---

## Practical Work 3 – Multimodal Gesture Interaction

Development of a virtual drawing application controlled entirely by hand gestures.

### Technologies

- Unity
- Python
- MediaPipe
- UDP Communication
- Computer Vision

### Main Features

- Real-time hand tracking
- Gesture recognition
- Drawing with finger movement
- Brush thickness adjustment using pinch gesture
- Color selection
- Brush/Eraser mode
- Live communication between Python and Unity

### Focus

Computer vision, multimodal interaction, networking and natural user interfaces.

# Final Project – Gesture-Controlled Minecraft World

Integration of all concepts explored throughout the semester into a single interactive virtual environment.

## Concept

A procedural Minecraft-style world where the player interacts exclusively through hand gestures while being chased by an intelligent ML-Agent.

---

## Core Functionalities

- Procedural voxel world generation
- Terrain generation using Perlin Noise
- Trees, lakes and natural terrain
- Hand gesture player controls
- Real-time Python ↔ Unity communication
- ML-Agent trained using PPO
- Block placement and destruction
- Gesture-controlled navigation
- Intelligent enemy pursuit
